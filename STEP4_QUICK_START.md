# 第四步"换样式"快速启动指南

### 1. 启动应用

```bash
npm run dev
```

### 2. 进入第四步

1. 在首页点击"开始设计"
2. 完成**第1步：填信息**（填写窗户尺寸等基本信息）

5. 点击"下一步"进入**第4步：换样式**

## 功能概览

### 🎨 统一配置模式

**一次性配置所有窗扇，快速统一风格**

#### 材质配置
- **室内材质**：类型（断桥铝/实木/塑钢/复合材料）+ 系列 + 颜色 + 表面处理
- **室外材质**：同上，或一键复制室内材质

#### 玻璃配置
- **类型**：钢化/夹胶/Low-E/双层中空/三层中空
- **规格**：根据类型自动调整（如5+12A+5）
- **镀膜**：Low-E/阳光控制/自清洁
- **透光率**：30%-90%可调
- **隔音等级**：25-45dB可调

#### 执手配置
- **样式**：标准/带锁/折叠/双执手
- **材质**：铝合金/锌合金/不锈钢
- **颜色**：6种可选
- **离地高度**：900-1500mm（推荐1200mm）
- **位置**：左侧/右侧/中间

#### 纱网配置
- **类型**：玻璃纤维/不锈钢/金刚网/隐形
- **材质**：根据类型自动调整
- **颜色**：灰色/黑色/白色/银色
- **网眼尺寸**：0.6-1.2mm
- **开启方式**：固定式/推拉式/可收缩式

### 🎯 单一配置模式

**逐个配置每个窗扇，实现差异化设计**

1. 选择要配置的扇（F1, F2, F3, F4...）
2. 根据扇类型配置：
   - **开启扇**：玻璃 + 执手
   - **固定扇**：仅玻璃

### 🪟 3D实时预览

**右侧实时查看3D窗户模型**

- **旋转**：鼠标左键拖拽
- **缩放**：鼠标滚轮 或 使用+/-按钮
- **平移**：鼠标右键拖拽（或Shift+左键）
- **重置视角**：点击"重置"按钮
- **线框模式**：切换查看结构框架

## 常用操作

### 保存配置

1. 配置完成后，点击底部**"保存配置"**按钮
2. 配置会保存到浏览器本地存储
3. 下次可以直接加载

### 加载配置

1. 点击底部**"加载已保存配置"**按钮
2. 之前保存的配置会立即恢复

### 重置配置

1. 点击底部**"重置"**按钮
2. 所有配置恢复为默认值

## 常见问题

### Q: 配置保存在哪里？
A: 配置保存在浏览器的LocalStorage中，仅在当前浏览器有效。

### Q: 可以导出配置吗？
A: 当前版本暂不支持导出，未来版本会添加。

### Q: 3D模型不显示怎么办？
A: 检查浏览器是否支持WebGL，或尝试更新浏览器版本。

### Q: 单一配置会覆盖统一配置吗？
A: 当前版本单一配置是基于统一配置的，未来会支持独立配置。

### Q: 如何查看不同角度的窗户？
A: 使用鼠标拖拽旋转3D模型，或使用重置按钮回到默认视角。
