## 技术实现

### 图片加载优化
```tsx
// 门窗图片展示
<img 
  src={window.image || "/placeholder.jpg"} 
  alt={window.type}
  className="w-full h-full object-cover hover:scale-105 transition-transform duration-300"
/>
```

### 响应式图片
- 使用 `aspect-square` 保证图片比例
- `object-cover` 确保图片填充容器
- `overflow-hidden` 防止缩放溢出
- `transition-transform` 平滑过渡动画

### 性能优化
- 图片懒加载（Next.js 自动优化）
- 合理的图片尺寸
- 占位图备用方案
- 渐进式图片加载
