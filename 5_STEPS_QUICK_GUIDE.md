# 门窗设计工具 - 5步骤快速指南

## 🎯 设计流程概览

新的5步骤设计流程让门窗设计更加简单、高效！

```
步骤1: 基础选型 → 步骤2: 选择窗型 → 步骤3: 窗型设计 → 步骤4: 配置选择 → 步骤5: 成果输出
   ↓              ↓              ↓              ↓              ↓
户型信息       选择结构        详细设计        材质参数        报价效果图
```

---

## 📝 详细步骤说明

### 第1步：基础选型 ✨

**目标**: 输入基础信息，获取智能推荐

**操作指引**:
1. **AI智能识别** (可选)
   - 点击"去上传"按钮
   - 拍照或上传现场照片
   - AI自动识别尺寸信息

2. **手动输入窗型信息**
   - 输入宽度和高度 (单位: mm)
   - 选择房屋类型：公寓、别墅、办公楼等
   - 选择安装位置：客厅、卧室、书房等
   - 选择楼层高度：低层、中层、高层等

3. **预算与推荐**
   - 快速选择预算范围：3万、5万、10万
   - 查看智能推荐的系列
   - 点击推荐卡片选择合适的系列

**提示**:
- 尺寸信息会影响后续设计
- 楼层高度影响风压要求
- 预算范围影响系列推荐

**完成标准**:
- ✓ 已输入宽度和高度
- ✓ 已选择房屋类型和位置
- ✓ 已选择推荐系列

---

### 第2步：选择窗型 🪟 ⭐ 新增

**目标**: 从窗型库中选择合适的窗型结构

**操作指引**:
1. **浏览窗型库**
   - 查看10种常用窗型配置
   - 每个窗型显示可视化预览
   - 显示分格数量和特点

2. **使用标签筛选**
   - 点击标签快速筛选：简洁、经济、通风等
   - 点击"展开"查看更多标签
   - 点击"全部"显示所有窗型

3. **选择窗型**
   - 点击窗型卡片进行选择
   - 选中的窗型会显示对勾标记
   - 顶部显示已选窗型名称
   - 选择后自动进入下一步

**窗型选项**:

| 窗型编号 | 分格说明 | 适用场景 | 特点 |
|---------|---------|---------|------|
| 001 | 1分格 | 小窗户 | 简洁、经济 |
| 002 | 上下2格 | 标准窗 | 通风灵活 |
| 003 | 左右2格 | 推拉窗 | 经典推拉 |
| 004 | 上中下3格 | 高窗 | 三层通风 |
| 005 | 左中右3格 | 大窗 | 三扇推拉 |
| 006/007 | 不均分3格 | 特殊需求 | 上下灵活 |
| 010 | 田字4格 | 标准窗 | 均衡对称 |
| 019 | 横向4格 | 超大窗 | 四扇推拉 |
| 031 | 6分格 | 大面积 | 对称均衡 |

**提示**:
- 选择与实际需求匹配的分格数量
- 考虑开启方式和通风需求
- 后续步骤可以继续调整

**完成标准**:
- ✓ 已选择一个窗型
- ✓ 自动进入步骤3

---

### 第3步：窗型设计 🎨

**目标**: 在选定窗型基础上进行详细设计

**操作指引**:
1. **查看初始布局**
   - 系统自动加载步骤2选择的窗型
   - 梃（分隔条）已自动布局
   - 区域已自动划分

2. **调整梃位置** (可选)
   - 点击梃可以删除
   - 点击"➕竖梃"或"➕横梃"进入添加模式
   - 在画布上点击添加新梃
   - 梃会自动重新计算区域

3. **配置区域**
   - 点击画布上的区域进行选择
   - 左侧面板显示区域配置选项
   - 设置区域类型：封窗 / 开启窗
   - 选择开启方式：平开、推拉、内倒等
   - 选择开启方向：左开、右开、上开、下开
   - 选择玻璃类型：单层、双层中空、Low-E等
   - 添加配件：纱窗、格栅、护栏

4. **缩放和查看**
   - 使用缩放按钮调整显示比例
   - 鼠标滚轮 + Ctrl/Cmd 也可缩放
   - 触摸屏支持双指缩放

**快捷操作**:
- 点击"切换窗型"返回步骤2重新选择
- 区域编号标识清晰显示
- 选中区域高亮蓝色边框

**配置说明**:

**区域类型**:
- 🔒 **封窗**: 固定不开启，适合观景
- 🚪 **开启窗**: 可开启通风，需配置开启方式

**开启方式**:
- **平开**: 向内或向外开启，密封性好
- **推拉**: 左右推拉，节省空间
- **内倒**: 上部内倾，安全通风
- **固定**: 不开启，仅通光

**玻璃类型**:
- **单层玻璃** (5mm): 经济型
- **双层中空** (5+9A+5): 隔音保温
- **Low-E玻璃** (5+12A+5): 节能隔热高端
- **夹胶玻璃** (5+0.76PVB+5): 安全隔音防爆

**提示**:
- 高层建议选择推拉窗，安全性高
- 卧室建议配置隔音玻璃
- 厨房卫生间建议平开窗，通风好

**完成标准**:
- ✓ 梃位置确定
- ✓ 所有区域配置完成
- ✓ 开启方式和玻璃类型选定

---

### 第4步：配置选择 ⚙️

**目标**: 选择材质和参数设置

**功能说明**:
- 全局配置：统一设置材质、玻璃、五金
- 局部微调：单独调整特定区域参数
- (开发中，更多功能即将推出)

**提示**:
- 此步骤即将完善更多功能
- 建议使用全局配置提高效率

---

### 第5步：成果输出 📊

**目标**: 生成报价单和效果图

**功能模块**:

**1. 报价单生成**
- 查看预估总价
- 查看详细费用明细：
  - 型材费用
  - 玻璃费用
  - 五金费用
  - 安装费用
- 下载详细报价单

**2. 效果图生成**
- 上传现场照片（拍照或相册）
- AI自动合成设计效果
- 生成高质量效果图
- 下载效果图用于方案展示

**提示**:
- 现场照片越清晰，效果图越真实
- 支持多个角度的效果图
- 可用于客户沟通和方案汇报

**完成标准**:
- ✓ 查看报价信息
- ✓ 下载报价单（可选）
- ✓ 生成效果图（可选）

## 🎯 使用技巧

### 快速上手
1. 第一次使用建议按流程完整走一遍
2. 每个步骤都有清晰的提示和说明
3. 不确定时可以使用推荐选项

### 效率提升
1. 使用AI照片识别快速获取尺寸
2. 使用标签筛选快速找到合适窗型
3. 选择预设窗型后微调，而不是从头设计

### 注意事项
1. 宽度和高度必须填写
2. 选择窗型后可以继续调整
3. 开启窗建议配置双层玻璃
4. 高层建议加强风压等级

## ❓ 常见问题

**Q1: 可以重新选择窗型吗？**
A1: 可以。在步骤3点击"切换窗型"按钮返回步骤2重新选择。

**Q2: 选择窗型后必须使用吗？**
A2: 不是。步骤3可以继续调整梃的位置和区域配置。

**Q3: 能自定义窗型吗？**
A3: 可以。在步骤3可以添加或删除梃，完全自定义窗型结构。

**Q4: 区域配置可以批量设置吗？**
A4: 目前需要逐个配置，批量配置功能正在开发中。

**Q5: 报价是实时的吗？**
A5: 报价是基于配置自动计算的参考价，实际价格需要确认。
