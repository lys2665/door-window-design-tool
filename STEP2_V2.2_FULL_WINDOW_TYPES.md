## 🔧 技术特性

### 1. 基于比例的梃配置
```typescript
// 简单梃
{ type: 'vertical', ratio: 0.5 }

// 部分梃（指定范围）
{ type: 'vertical', ratio: 0.5, startRow: 0.5 }  // 仅下半部分
{ type: 'horizontal', ratio: 0.5, endCol: 0.333 } // 仅左1/3部分
```

### 2. 复杂梃配置示例

**017窗型（上3格，下1格）**：
```typescript
mullions: [
  { type: 'horizontal', ratio: 0.25 },              // 第1条横梃
  { type: 'horizontal', ratio: 0.5 },               // 第2条横梃
  { type: 'horizontal', ratio: 0.75 },              // 第3条横梃
  { type: 'vertical', ratio: 0.333, endRow: 0.75 }, // 竖梃1（上3/4）
  { type: 'vertical', ratio: 0.666, endRow: 0.75 }  // 竖梃2（上3/4）
]
```

**022窗型（左2中1右2）**：
```typescript
mullions: [
  { type: 'vertical', ratio: 0.333 },                // 竖梃1
  { type: 'vertical', ratio: 0.666 },                // 竖梃2
  { type: 'horizontal', ratio: 0.5, endCol: 0.333 },  // 横梃（左侧）
  { type: 'horizontal', ratio: 0.5, startCol: 0.666 } // 横梃（右侧）
]
```

## 📐 尺寸自适应

### 基于框架尺寸计算

**示例1：2400×1800mm框架，选择003窗型**
```
竖梃位置 = 2400 × 0.5 = 1200mm
区域1: x=0, width=1200mm
区域2: x=1200, width=1200mm
```

**示例2：3000×2400mm框架，选择027窗型**
```
横梃位置 = 2400 × 0.5 = 1200mm
竖梃1位置 = 3000 × 0.333 = 1000mm
竖梃2位置 = 3000 × 0.666 = 2000mm
竖梃3位置 = 3000 × 0.5 = 1500mm（仅下半部分）
→ 自动生成5个区域
```

## 🎮 使用流程

1. **Step1**：输入框架尺寸（如 2400×1800）
2. **Step2**：浏览32种窗型
3. **选择窗型**：点击合适的窗型卡片
4. **自动生成**：系统根据梃配置自动计算区域
5. **配置区域**：点击区域设置封窗/开启窗等属性
6. **自定义调整**：继续添加/删除梃进行微调
