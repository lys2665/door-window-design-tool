# Step2 窗型设计 - 快速开始指南

## 🚀 5分钟快速上手

### 访问路径
```
http://localhost:3000/design
```

### 快速体验流程

#### 第1步：完成基础信息（Step1）

1. 访问设计页面
2. 输入门窗尺寸（或使用默认值）：
   - 宽度：2400mm
   - 高度：1800mm
3. 选择其他基础信息
4. 点击"下一步"进入 Step2

---

#### 第2步：选择窗型

进入 Step2 后，会看到 **6种窗型推荐**：

**推荐选择：**
- 新手：选择"推拉窗 2扇"（最简单）
- 进阶：选择"推拉窗 3扇"（常用）
- 高级：选择"组合窗"（功能全）

**操作：** 点击任意窗型卡片

---

#### 第3步：了解主界面布局

选择窗型后进入主设计界面，布局如下：

```
┌────────────────────────────────────────────┐
│ [切换窗型]         尺寸: 2400×1800mm      │ ← 顶部工具栏
├──────┬───────────────────────┬─────────────┤
│      │                       │             │
│ 俯   │      2D画布           │  配件面板   │
│ 视   │   （点击窗扇编辑）    │ （选中后显示）│
│ 图   │                       │             │
│      │                       │             │
└──────┴───────────────────────┴─────────────┘
```

---

#### 第4步：选中并编辑窗扇

**方式一：在画布中点击**
- 直接点击中间画布的任意窗扇
- 选中的窗扇会显示蓝色边框

**方式二：在俯视图点击**
- 点击左侧俯视图的小方格
- 画布会同步选中对应窗扇

**选中后的效果：**
- 窗扇边框变蓝
- 显示尺寸标注
- 右上角出现操作按钮
- 右侧出现配件设置面板

---

#### 第5步：基础操作演示

**📏 调整尺寸**

*方法1：右侧面板输入*
```
1. 选中窗扇
2. 在右侧找到"宽度"和"高度"输入框
3. 输入新的数值（单位：mm）
4. 画布立即更新
```

*方法2：拖拽调整*
```
1. 选中窗扇
2. 找到右下角的小圆点
3. 拖拽调整大小
```

---

**🪟 设置为开启扇**

```
1. 选中窗扇
2. 点击右上角的"设置"按钮（齿轮图标）
3. 或在右侧面板点击"窗扇类型"按钮
4. 窗扇变为蓝色背景，显示把手图标
```

---

**📋 复制窗扇**

```
1. 选中窗扇
2. 点击右上角的"复制"按钮
3. 生成一个相同的窗扇（稍微偏移）
4. 可以继续调整新窗扇
```

---

**⚙️ 配置执手（仅开启扇）**

设置为开启扇后，在右侧面板：

```
1. 调整执手高度
   - 滑动条：拖动快速调整
   - 输入框：精确输入（建议900-1100mm）

2. 选择执手样式
   - 🔧 拨叉执手（常用）
   - 🔐 两点锁执手（安全）
   - ⭕ 圆柄执手（美观）
   - ▬ 平柄执手（简约）
```

---

**🔧 选择玻璃类型**

所有窗扇都需要选择玻璃：

```
推荐选择：
- 住宅：双层中空玻璃（隔音+保温）
- 高端：Low-E玻璃（节能+隔热）
- 安全：夹胶玻璃（防盗+隔音）
- 经济：单层玻璃
```

---

**🛠️ 添加配件（可选）**

在右侧面板底部：

```
常用配件：
1. 护栏 - 阳台必备，选择铝合金材质
2. 纱窗 - 开启扇推荐，选择金刚网
3. 隔栅 - 装饰性，选择竖条样式
4. 新风系统 - 通风换气，选择隐藏式
```

### 💡 实用技巧

#### 技巧1：快速定位窗扇
使用左侧俯视图快速找到要编辑的窗扇，比在大画布中点击更精准。

#### 技巧2：合理设置开启扇
- 推拉窗：通常一半窗扇可开启
- 平开窗：全部窗扇可开启
- 固定窗：不设置开启扇

#### 技巧3：执手高度参考
- 成人：1000-1100mm（推荐）
- 儿童房：900-950mm
- 老人房：950-1000mm

#### 技巧4：使用复制功能
如果多个窗扇配置相同，配置好一个后直接复制，省时省力。

#### 技巧5：随时切换窗型
如果发现当前窗型不合适，点击"切换窗型"重新选择，不会丢失尺寸数据。

### 🎯 完整示例：配置客厅推拉窗

**场景：客厅阳台推拉窗，2400×1800mm**

```
1️⃣ 选择窗型
   → 推拉窗 3扇

2️⃣ 设置左边窗扇
   → 选中 → 设为开启扇
   → 执手高度：1000mm
   → 执手样式：拨叉执手
   → 纱窗：金刚网
   → 玻璃：双层中空

3️⃣ 设置中间窗扇
   → 选中 → 设为开启扇
   → 执手高度：1000mm
   → 执手样式：拨叉执手
   → 纱窗：金刚网
   → 玻璃：双层中空

4️⃣ 设置右边窗扇
   → 选中 → 保持固定扇
   → 玻璃：双层中空

5️⃣ 添加配件
   → 护栏（铝合金、直杆）
   → 下降料（铝合金、静音型）

6️⃣ 检查整体
   → 查看俯视图，确认布局合理
   → 确认开启扇位置方便使用

7️⃣ 完成
   → 点击"下一步"进入配置选择
```

**完成效果：**
- 3扇推拉窗
- 2扇可开启（左、中）
- 1扇固定（右）
- 配置了纱窗和护栏
- 总面积：约 4.32 ㎡

### ⚠️ 常见问题

**Q1: 为什么看不到配件设置面板？**
> A: 需要先选中一个窗扇，面板才会显示

**Q2: 如何删除窗扇？**
> A: 暂不支持删除，可以通过切换窗型重新开始

**Q3: 尺寸输入后没有变化？**
> A: 检查是否选中了正确的窗扇，确保输入的是数字

**Q4: 执手设置在哪里？**
> A: 只有开启扇才有执手设置，先设置为开启扇

**Q5: 如何返回重新选择窗型？**
> A: 点击顶部的"切换窗型"按钮

**Q6: 俯视图和画布不同步？**
> A: 这是一个 bug，刷新页面即可解决（请反馈）

### 🎨 设计建议

#### 通风考虑
- 推拉窗：至少50%窗扇可开启
- 平开窗：建议全部可开启
- 固定窗：搭配其他通风方式

#### 采光优化
- 落地窗：最大化采光
- 大面积玻璃：使用Low-E玻璃降低热量
- 避免过多隔条：影响视野

#### 安全性
- 高层建筑：必须安装护栏
- 儿童房：开启窗限位器
- 大面积玻璃：建议使用夹胶玻璃

#### 经济性
- 标准尺寸：更经济
- 合理配件：避免过度配置
- 材质选择：根据需求而定
