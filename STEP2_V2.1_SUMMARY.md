## 🔑 核心特性

### 1️⃣ 基于框架尺寸
- **框架来源**：Step1 用户输入的宽高
- **自动计算**：梃的实际位置 = 框架尺寸 × 比例
- **示例**：框架 2400×1800，窗型003（竖梃50%）→ 梃位置 1200mm

### 2️⃣ 支持部分梃
- **全长梃**：贯穿整个窗框
- **部分梃**：仅覆盖特定区域
- **示例**：006窗型的竖梃仅在下半部分

### 3️⃣ 智能区域生成
- 根据梃的布局自动计算所有区域
- 区域尺寸精确无误
- 无需手动调整

## 📐 梃配置示例

### 简单窗型（003）
```javascript
mullions: [
  { type: 'vertical', ratio: 0.5 }  // 竖梃在中间
]
```

### 复杂窗型（006）
```javascript
mullions: [
  { type: 'horizontal', ratio: 0.5 },          // 横梃在中间
  { type: 'vertical', ratio: 0.5, startRow: 0.5 }  // 竖梃仅下半部分
]
```
